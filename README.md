# 📚 BookStore– MERN Book Store

A full-stack Book Store web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). The platform provides a seamless experience for discovering, reading, and purchasing books with secure authentication and protected premium content.

##  Features

* User Authentication (JWT)
* Free Books available for all users
* Premium Books is protected without login you cant access
* Protected Routes for premium content
* Responsive UI with Tailwind CSS
* RESTful API Integration

## 🛠️ Tech Stack

**Frontend**

* React.js
* Tailwind CSS

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB

**Authentication**

* JWT (JSON Web Token)

## 🔒 Access Control
* **Free Books:** Available to all users without payment.
* **Premium Books:** Accessible only to authenticated users who have successfully purchased the book.
* Protected routes ensure unauthorized users cannot access premium content.

## 📈 Key Highlights
* Dynamic and responsive user interface
* Secure authentication and authorization
* Scalable REST API architecture
* MongoDB database integration
* Clean and modular project structure
